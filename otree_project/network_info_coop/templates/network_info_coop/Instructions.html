Instructions

You are in a group of {{ n }} players for {{ num_rounds }} rounds.

1) Hidden state and signals
  • At the start, a hidden state is randomly chosen: HIGH or LOW. It remains fixed throughout all rounds.
  • Each round, you receive a private signal about the state.
  • Signals are noisy: they match the true state with probability {{ signal_accuracy }}.
Important: Signals are generated by the system. Players cannot change, hide, or manipulate signals.
2) Network structure (who you observe)

Network structure: {% if network_type == 'hub' %} Hub-and-Spoke (centralised information) {% else %} Ring (decentralised information) {% endif %}

{% if network_type == 'ring' %}

Ring network rule:

  • You observe your own signal.
  • You also observe the signals of your two neighbours (left and right).
  • No single player observes everyone.
{% else %}

Hub-and-Spoke network rule:

  • Player 1 is the hub (central player).
  • The hub observes all players’ signals each round.
  • Spokes (Players 2–{{ n }}) observe their own signal and the hub’s signal only.
  • Spokes do not observe other spokes’ signals.
Note: If you are Player 1, you will receive more information than others because of the network structure.
{% endif %}
3) Decisions each round
  • After viewing signals, you will decide whether to contribute to a group project.
  • You will also report your belief (0–100) that the state is HIGH.
  • Beliefs do not affect payoffs and are not shown to other players.
Payoff reminder: Each round you receive an endowment of {{ endowment }}. If you contribute, you contribute {{ contrib_amt }}.
